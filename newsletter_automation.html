Analysis Newsletter Automation

Automation of analysis newsletters in full requiring only to execute a code in R and adding a macro

Overview

In order to automate the elaboration of a analysis newsletters that are published periodically to save time and avoid typographical errors, solutions were developed so that the entire fact sheet could be prepared without the need to write or make graphs, requiring only running a code in R, adding a macro to the Excel file generated and executing it. This generates a Word document with the analysis and graphs generated according to the latest information published by INEGI for different surveys.
The newsletters were written in Spanish for the Instituto de Información Estadística y Geográfica de Jalisco (IIEG).

How to use the Files

  1. Download the necessary R packages.
  2. Change the directory in R archive to one fo your choice.
  3. Run the entire R code (Ctrl A + Ctrl Enter).
  4. Open the Excel file created by the code in the previously specified directory.
  5. In the Excel file Go to Programmer > Code > Visual Basic.
  6. In the VBAProject area right click and select "Import file" and select the Macro (.bas file).
  7. Change the directory in the .bas file where the created document will be saved and the locations where the Charts and Word templates will be taken from.
  8. Run the macro with Ctrl + A and it will automatically make the graphs, which will be taken to a Word document where it will write all the analysis, titles, sources and notes in the appropriate format.
